# AB Events — موقع تصميم وتنفيذ الفعاليات الفاخرة

موقع فاخر (Luxury / Cinematic / Awwwards-style) لوكالة **AB Events** المتخصّصة في تنظيم الفعاليات في **الطائف**.

الرابط بعد النشر (GitHub Pages): `/<repo>/ab-events/`
الملف الرئيسي: [`index.html`](./index.html) — صفحة واحدة مكتفية ذاتياً (HTML + CSS + JS، بدون build).

---

## 1) تحليل الموقع المرجعي (ofuqeventssa.com)

| المحور | الملاحظة | كيف طوّرناه في AB Events |
|---|---|---|
| **الهيكل** | Landing page واحدة: Hero → إحصائيات → خدمات → طريقة العمل → أعمال → تواصل | وسّعناه إلى ١٢ قسماً: أضفنا Portfolio بفلترة + Modal، عملاء، فيديوهات، Gallery + Lightbox، آراء، FAQ، خريطة |
| **UX** | بسيط وواضح، تركيز على الحجز عبر واتساب | حافظنا على البساطة + أضفنا Smooth scroll، Scroll reveal، Loading screen، Back-to-top، زر واتساب عائم |
| **UI** | نظيف، ألوان محايدة + ذهبي، صور فعاليات | رفعناه إلى Obsidian & Gold — أسود فاخر + ذهبي شامبين، تباين أقسام داكنة/فاتحة بأسلوب Apple |
| **الألوان** | أبيض/رمادي + ذهبي ناعم | باليت أسود عميق `#0B0B0D` + ذهبي `#C9A24B` + شامبين `#E4CD8F` + آيفوري `#F5F2EC` |
| **الخطوط** | Sans عربي حديث | Alexandria (عربي) + Manrope (لاتيني/أرقام) + Cormorant Garamond (عناوين مائلة فاخرة) |
| **الحركة** | خفيفة | Scroll reveal, Animated counters, Gradient drift, Hover micro-interactions, Custom cursor, Page rings |
| **الخدمات** | ٦ بطاقات | ٩ بطاقات بتأثير Hover داكن + ترقيم |
| **المشاريع** | معرض صور بسيط | Masonry Grid + فلترة تصنيفات + Modal تفاصيل (عميل/موقع/تاريخ/خدمات) |

## 2) تحليل حساب TikTok (@abevents02)

- **الهوية:** «AB Events — تنسيق حفلات | الطائف» (وكالة تنظيم فعاليات/حفلات).
- **نوع الفعاليات:** حفلات خاصة (تخرّج، أعياد ميلاد، استقبال مواليد)، مناسبات وتنسيقات راقية.
- **الفئة المستهدفة:** العائلات والأفراد في الطائف والمنطقة الغربية الباحثون عن تنسيق فاخر.
- **الأسلوب:** لقطات كواليس وتنفيذ، ألوان دافئة راقية، مونتاج قصير سريع الإيقاع.

> **ملاحظة تقنية:** TikTok يمنع القراءة الآلية لصفحات الملفات الشخصية، لذلك بدل تخمين المحتوى، **ضمّنّا الفيديوهات الحقيقية مباشرة** عبر Creator Embed الرسمي في قسم «الفيديوهات» — بحيث يعرض الموقع أحدث مقاطعهم تلقائياً دائماً. هذه أدقّ طريقة لإبراز أعمالهم الفعلية.

**اقتراح الترتيب داخل الموقع:** أفضل اللقطات (لحظات الكشف/الريفيل، اللقطات الواسعة للديكور، تفاصيل الطاولات) تُستخدم لاحقاً كـ: (1) فيديو Hero سينمائي، (2) أغلفة بطاقات Portfolio، (3) صور Gallery. الأماكن جاهزة في الكود للاستبدال.

- **الاسم/الشعار:** مونوغرام `AB` بخط Cormorant داخل مربّع أسود بحد ذهبي.
- **الباليت:**
  - Obsidian `#0B0B0D` · Ink-2 `#111116`
  - Gold `#C9A24B` · Champagne `#E4CD8F` · Gold-dark `#9A7826`
  - Ivory `#F5F2EC` · Cream `#FBF9F4` · Gray `#8A8A92`
- **الطباعة:** Alexandria / Manrope / Cormorant Garamond.
- **النبرة:** فاخرة، سينمائية، مختصرة، واثقة.

## 6) الإعداد (مهم)

عدّل ثوابت التواصل أعلى وسم `<script>` في `index.html`:

```js
const CONFIG = {
  whatsapp: "9665XXXXXXXX",   // رقم واتساب بصيغة دولية بدون +
  phone:    "+9665XXXXXXXX",  // رقم الاتصال
};
```

- استبدل الأغلفة/الشعارات/الآراء التوضيحية (المعلّمة بـ «*») بمحتوى AB Events الحقيقي.
- لإضافة فيديو Hero حقيقي: ضع `hero.mp4` وأضف `<video autoplay muted loop playsinline>` مكان `.hero-stage`.

## 7) خارطة طريق لوحة التحكم (CMS) — مرحلة تالية

لوحة تحكم حقيقية (إدارة مشاريع/خدمات/صور/فيديو/عملاء/آراء/فريق/إحصائيات/FAQ/SEO/مستخدمين + Audit Log + رفع بالسحب والإفلات + معاينة مباشرة) تتطلّب **باك-إند** ولا يمكن أن تعمل كصفحة ثابتة على GitHub Pages. المعمارية المقترحة:

- **الواجهة:** Next.js + TypeScript + Tailwind + shadcn/ui (ترقية طبيعية لهذا التصميم).
- **الباك-إند/التخزين:** Supabase (Auth + Postgres + Storage) — موجود مسبقاً في هذا المستودع ضمن `dhawi-store`.
- **المكوّنات:** جداول محتوى + رفع صور إلى Storage + سجل تعديلات (audit_log) + صلاحيات (RLS).

جاهز للتنفيذ في مرحلة لاحقة عند الطلب.
