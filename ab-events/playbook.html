الهوية البصرية

تطوير هوية AB لمستوى عالمي — مع الحفاظ على شعارك الحالي A|B.

🔠 الشعار

  • احتفظ بالمونوغرام A|B (أنيق)
  • اصنع نسخاً: أفقي، رأسي، أيقونة، مائي شفاف
  • watermark صغير بزاوية بدل الوسط

🎨 الألوان

  • أساسي: أوبسيديان + ذهبي شامبين
  • توقيع: تراكوتا (ورد الطائف) + سيج (أوكالبتوس)
  • محايد: آيفوري + بيج دافئ

✍️ الخطوط

  • عرض: Cormorant (فخامة كلاسيكية)
  • عربي: Alexandria / نص واضح حديث
  • لاتيني: Manrope للأرقام والتفاصيل

📷 أسلوب التصوير

  • تدرّج لوني دافئ موحّد لكل الصور
  • لقطة «بطل» + ماكرو تفاصيل لكل مناسبة
  • مساحات فاضية = فخامة هادئة

🖼️ عرض الأعمال

  • قبل/بعد + فيديو قصير
  • تقسيم حسب المناسبة + فلترة
  • صور بملء الشاشة

📎 المطبوعات

  • بروفايل PDF فاخر + عرض سعر منسّق
  • بطاقة عمل + بطاقة شكر بعد المناسبة
  • قوالب سوشيال موحّدة

🖥️ تطوير الموقع (م٦)

الرئيسية

  • فيديو Hero سينمائي (بدل الخلفية المتحركة)
  • قبل/بعد بارز فوق
  • شعارات ثقة/تقييمات Google

الخدمات + الأسعار

  • صفحة باقات (٥ مستويات)
  • «يبدأ من» بدل السعر الكامل
  • زر حجز لكل باقة

الحجز والتواصل

  • نموذج ذكي (نوع/تاريخ/ميزانية)
  • خريطة + واتساب + اتصال
  • رد تلقائي فوري

الأداء والسيو

  • ضغط الصور (WebP) — أسرع تحميل
  • عناوين ووصف SEO لكل صفحة
  • Schema للأعمال المحلية (الطائف)

الجوال

  • أزرار كبيرة + واتساب عائم
  • معرض يسحب بالإصبع
  • سرعة أولاً

الثقة

  • «من نحن» بقصة حقيقية
  • تقييمات موثّقة
  • أرقام حقيقية (+مناسبة)

📷 الصور المرجعية (م٥)

لكل فكرة في المكتبة الإبداعية يوجد Prompt جاهز يولّد «نسختنا» مباشرة — أرقى من أي مرجع لأنها مضبوطة على هويتك (ورد الطائف، النجدي، الإضاءة الدافئة). القاعدة: نستلهم الأسلوب، ولا ننسخ أي تصميم محمي.
